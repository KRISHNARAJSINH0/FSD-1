25. String Methods:
        => Note : All string methods return a new value,They do not change the original string.
        =>      String length =>Returns the length of a string
                String charAt() =>Returns the character at a specified index (position)
                String charCodeAt() =>Returns the Unicode of the character at a specified index
                String codePointAt() =>Returns the Unicode value at an index (position) in a string
                String concat() =>Returns two or more joined strings
                String at() =>Returns an indexed character from a string
                String [.]
                String slice() =>Extracts a part of a string and returns a new string
                String substring() =>Extracts characters from a string, between two specified      indices (positions)

                String substr() =>Deprecated. Use substring() or slice() instead.
                String toUpperCase() =>Returns a string converted to uppercase letters
                String toLowerCase() =>Returns a string converted to lowercase letters
                String isWellFormed() =>Returns true if a string is well formed
                String toWellFormed() =>Returns a string where "lone surrogates" are replaced with the Unicode replacement character

                String trim() => Returns a string with removed whitespaces
                String trimStart() => Returns a string with removed whitespaces from the start
                String trimEnd() =>Returns a string with removed whitespaces from the end
                String padStart() =>Pads a string from the start
                String padEnd() =>Pads a string at the end
                String repeat() =>Returns a new string with a number of copies of a string
                String replace() =>Searches a string for a pattern, and returns a string where the first match is replaced

                String replaceAll() =>Searches a string for a pattern and returns a new string where all matches are replaced

                String split() =>Splits a string into an array of substrings

                String lastIndexOf() => Returns the index (position) of the last occurrence of a value in a string

                String search() => Searches a string for a value, or regular expression, and returns the index (position) of the match

                String match() =>Searches a string for a value, or a regular expression, and returns the matches

                String matchAll() =>Searches a string for a value, or a regular expression, and returns the matches

                String includes() =>Returns if a string contains a specified value
                String startsWith() =>Checks whether a string begins with specified characters
